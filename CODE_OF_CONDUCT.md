# Code of conduct

Weft is an owner-led experimental project. Participants are expected to be
respectful, constructive, and professional. Harassment, discrimination,
threats, personal attacks, and publication of private information are not
acceptable.

If a discussion or contribution space is enabled, report concerns privately to
the repository owner through the associated GitHub profile. The owner decides
how to investigate and respond, including moderation or removal from the
project space.

This policy will be expanded if the project develops an active contributor
community.
