# AboimPinto.Weft.Abstractions

Experimental metadata contracts for Weft HTML-first .NET feature modules.

This package contains the attributes and descriptors used to declare features,
routes, actions, and execution modes. Most applications should reference
`AboimPinto.Weft.Server`, which brings this package transitively.

The API is experimental and may change before a stable release.

Project: <https://github.com/aboimpinto/weft-dotnet>
