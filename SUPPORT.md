# Support and feedback

Weft is not production-ready and has no support commitment yet.

Before public issue or discussion channels are enabled:

- use the repository owner’s GitHub profile for design feedback;
- do not report suspected vulnerabilities publicly; see [SECURITY.md](SECURITY.md);
- do not submit code contributions until a public license and contribution
  terms are in place.

Once implementation begins, the repository will define which channels are for
design discussion, reproducible defects, feature proposals, and security
reports.
